# ReghTechLab Portfolio

Welcome to the documentation for **ReghTechLab Portfolio**, a full-stack web application built with Python, Flask, SQLAlchemy, and Tailwind CSS.

## Overview

The ReghTechLab portfolio is designed to showcase projects, host a fully functional blog, and manage a community of users who can interact through comments and likes.

It includes:

- **Authentication System:** Secure registration and login.
- **Blog Engine:** Features for creating, publishing, and interacting with posts.
- **Modern UI:** Designed with Tailwind CSS for a responsive, sleek experience.
